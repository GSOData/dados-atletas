# Projeto Certificação 2 - Dados dos Atletas 🏃‍♂️

Este projeto foi desenvolvido como parte da certificação do curso **DEVstart**. O objetivo é aplicar conceitos de **Programação Orientada a Objetos (POO)** em JavaScript para gerenciar informações de atletas e automatizar cálculos de performance.

## 📌 Sobre o Projeto
A aplicação recebe os dados de um atleta (nome, idade, peso, altura e notas) e, através de uma classe organizada, realiza os seguintes cálculos:
- **Categoria:** Define se o atleta é Infantil, Juvenil, Intermediário ou Adulto com base na idade.
- **IMC:** Calcula o Índice de Massa Corporal.
- **Média Válida:** Calcula a média das notas descartando a maior e a menor (conforme regra de competições oficiais).
